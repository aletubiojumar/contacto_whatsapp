# Carpeta de datos

Almacena archivos locales `*.jsonl` con la cola de notas a procesar. Añade cada línea como un JSON con `codigo`, `modelo` y `texto`.

> ⚠️ Los archivos reales están ignorados por `.gitignore` (`data/*.jsonl`). Sólo sube ejemplos sintéticos si lo necesitas.

## Ejemplo rápido

```jsonl
{"codigo": "657356095", "modelo": "241", "texto": "Texto de la nota 1"}
{"codigo": "123456789", "modelo": "9151", "texto": "Texto de la nota 2"}
```
